# Mundial Stickers — Landing Page

Sitio oficial de **Mundial Stickers**, app móvil offline para coleccionistas del Mundial 2026.

🌐 **Sitio web**: https://amsoftpe.github.io/mundial-stickers-web/

## Sobre Mundial Stickers

App móvil para registrar tu inventario del álbum del Mundial y hacer intercambios cara a cara con otros coleccionistas mediante códigos QR. Funciona 100% offline, sin servidores, sin cuentas.

- `index.html` — Landing page principal.
- `privacy.html` — Política de privacidad oficial.
